About SubX

SubX is a powerful subdomain takeover detection tool designed to help security researchers and bug bounty hunters identify potential vulnerabilities in web applications. Version 2.0 introduces several new features including enhanced service detection, improved reporting, and performance optimizations.

Quick Start Guide

  1. Navigate to the New Scan tab
  2. Enter a domain or upload a list of subdomains
  3. Configure scan options as needed
  4. Click Start Scan to begin
  5. View results in real-time as they appear
